To run this program, ensure the following:
- You have internet connection
- YahooFinance is not down
- You have Python 3 installed
- You have pip installed 

Step 1:
Type in command line: 
"
cd /directory/to/this/folder
pip install -r requirements.txt
"

Wait for it to finish downloading

Step 2:

Run the batchfile "run.bat"

Note: If windows says it is a virus, create a new .bat file, and copy paste the elements of run.bat.
Then run the new bat file
